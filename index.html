What is HTML?

HTML, or Hyper Text Markup Language, has been around for some time now and is the backbone for all web pages.
Chances are that if you have visited a web page you have most likely encountered HTML. The most up to date version
of HTML is HTML5, which was released in October of 2014. HTML 5 adds audio, video and canvas elements allowing
for multimedia integration.

Building a web page.



The basic layout of an HTML file

HTML uses an easy layout format to organize the content of the web page.
The basic components of an HTML file include

  • The Doctype (<!DOCTYPE html>) is an instruction to the web browser about what version of the markup language the page is written in.
  • The html tags (<html>) tell the web browser where your html 'code' begins and where it ends.
  • The Header (<head>) is the behind the scenes manager for your HTML web page.
    The header is where all your external library calls go as well as any setup you might do to the website.
  • The Body (<body>) is where the main contents of your web page goes. This is where the user will spend
    most of their time looking at, so don't be afraid to make it look nice!

Getting started

Before we begin you should decide on an enviornment to write your website with. In this tutorial I will be using
Sublime Text this is a lightweight text editor that highlights code very nicely.
This editor is not required though so feel free to choose whatever enviornment you like.

Your first website

Now that you have your enviornment set up lets begin by setting up the basis for a web page.
The following code will set up what the Document Object Model, or DOM, will see as your very own web page.
Begin by typing this into a new text file:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>title</title>
    </head>
    <body>

    </body>
</html>

The above code is what we call an HTML shell. this is the basic layout for every web page.
go ahead and save your text file as index.html and open your website in a browser.
Pretty plain, right? Dont worry, we will be adding content at a later time. But for now congratulations on your first website!

Adding content to your web page:

Now that you have your website up and running its time to start adding content to your web page.
All of your editing will be done within the <body> tags for now. The <body> tag is
the front end that the user will see when visiting your web page. It is important to keep your code
orginized within this block as it can get pretty messy at times. Now, it's time to get some text onto the screen!
There are two basic components of html text, The title denoted by <h1> will display a big text
this is used to divide your website into different sections. Begin by adding <h1>Foo</h1>
into your body section of index.html this will add a title with the text foo to your web page.
This is great and all, but what about writing actual text that is not huge, this is done using the <p> tag.
This tag is also known as the paragraph tag allows the user to write several lines of text. This is useful for
getting information across to the user. As a matter of fact, the sentence you are currently reading is written within the
paragraph tag! Now lets add a paragraph to your website. Add the following line after your title <p>Bar</p>
Now, your website is starting to come togther! Feel free to add your own titles and paragraph.

Your HTML code should look like the following 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>title</title>
    </head>
    <body>

        <h1>Foo</h1>
        <p>Bar</p>
    
    </body>
</html>

Adding images to your web page:

Todays web pages are filled with images to try and catch visitors attention. Without images, among other things,
todays web would look very boring. Luckily, it is easy to add images to your page! To add an image you would use the
<img> tag. Try adding an image by adding the following line of code to your body:
<img src="http://bit.ly/1i8VVBA"/> Notice how this image code has a src field. This tells your DOM
where to look for the actual image. The src field can be a local path or a path to an image on the web. Now you can
make your web page look pretty gnarly!

Your HTML code should look like this: 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>title</title>
    </head>
    <body>
        <img src="http://bit.ly/1i8VVBA/>
    </body>
</html>

Adding links to your webpage:

As you can see, I have already added a link earlier in this tutorial, but who did I do that? Just like adding
images, adding a link is very easy, if not easier! To add a link, you would use the <a> tag.
This stands for an anchor tag and is the backbone for links on the web. Lets try try adding a link to your page!
Add the followig line of code: <a href="http://bit.ly/1LKaTfp">Link</a> Notice how we
are using href instead of src. href is similar to src, however it tells the web page to execute the link rather
than just pull information from it. Also, you can use href as a domain name or a local path. Now you can set up links
like a pro!

Your link code should look like this: 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>title</title>
    </head>
    <body>
        <a href="http://bit.ly/1LKaTfp">Link</a>
    </body>
</html>

Adding multimedia to your webpage:

Along with adding images to your web page, you may find it productive to add other multimedia content
such as audio and video. To add an audio file to your page you would use the <audio> tag.
It is worth noting that audio is not supported in all browsers.
To add video to your web page, you would use the <video> tag. It is also worth mentioning that
video is not supported in all browsers.

The following code implements both audio and video into the website using bar.mp3 and foo_bar.mp4 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>title</title>
    </head>
    <body>
       <!-- Audio -->
       <audio controls>
            <source src="bar.ogg" type="audio/ogg">
            <source src="bar.mp3" type="audio/mpeg">
            Your browser does not support the audio element.
        </audio>

        <!-- Video -->
        <video width="400" controls>
            <source src="foo_bar.mp4" type="video/mp4">
            <source src="foo_bar.ogg" type="video/ogg">
            Your browser does not support HTML5 video.
        </video>

    </body>
</html>
